# B_Looker

## Barrel Production Monitoring System

B_Looker is a smart production monitoring system developed for barrel manufacturing.

### Features

- QR Code Based Machine & Material Scanning
- Live Production Dashboard
- Chamber Tracking
- Rescan Detection
- WIP Monitoring
- Completed Job Tracking
- Google Sheets Integration
- Google Apps Script Backend
- Responsive Website

### Modules

- Home
- Operator
- Dashboard
- Reports
- About

### Technology

- HTML5
- CSS3
- JavaScript
- Google Apps Script
- Google Sheets
- GitHub Pages
